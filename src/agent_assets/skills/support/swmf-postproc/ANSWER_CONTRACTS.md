# Answer Contracts

Use these required sections in addition to the base `swmf-postproc` output contract.

## For `idl_inventory`
Always include:
- `authority_level`
- `entry_points`
- `helpers_or_supporting_routines`
- `source_paths`
- `verification_note`

## For `idl_usage_guidance`
Always include:
- `authoritative_procedure`
- `function_definition_evidence`
- `example_files`
- `assumptions`
- `copy_paste_ready_example`

## For `coupling_architecture_explanation`
Always include:
- `registry_components`
- `implemented_directed_links`
- `compile_time_gates`
- `runtime_enablement_note`
- `mermaid_diagram`

## For `artifact_comparison`
Always include:
- `compared_artifacts`
- `observed_differences`
- `impact_assessment`
- `missing_evidence`

## For `postprocess_failure_triage`
Always include:
- `observations`
- `conflicting_signals`
- `next_evidence_step`
- `skill_handoff`
